## github에 코딩파일 업로드

1. 내 프로필 Repositories 클릭
2. New 클릭
3. Repositories name에는 코딩파일 제목 작성
4. Create Repository 클릭
5. VS 코드 좌측 패널에서 git 아이콘 클릭
6. initialize Repository 클릭(파일을 깃으로 관리할거에요~ 라는 의미)
7. 업로드 할 파일만 + 눌러서 Staged Changes로 이동
8. Message 입력칸에 어떤 것이 변경되었는지 입력하고 Commit 버튼 클릭
9. 터미널에 순차적으로 복사 붙여넣기
git remote add origin https://github.com/simque15/git-study.git
git push -u origin main
